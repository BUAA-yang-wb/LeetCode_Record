## LeetCode 刷题记录

### 项目结构

```text
.
├── README.md       # 本项目的说明文档
└── hot_100/        # 100热题的解答和思路
```

### GPT答疑提示词

**第一轮对话初始提示词：**
```
我正在刷leetcode的100热题，为后面的找实习机试做准备。帮我解答后面的一些题目，并给出简要的解析。
```

**后续只需给GPT发送图片，即可直接给出答案及原理讲解;**

**最终将题解及解析汇总为一个.md文件：**

```
将上述题目的题解以及简要解答分析汇总为一个.md文件发给我。注意，时间复杂度说明和题目描述都不用写
```
