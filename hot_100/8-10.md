# LeetCode 热题题解汇总

## 3. 无重复字符的最长子串

### 核心思路

使用 **滑动窗口 + 哈希表**。

维护一个窗口 `[left, right]`，保证窗口内没有重复字符。右指针 `right` 不断向右遍历字符串，如果当前字符之前在窗口中出现过，就移动左指针 `left`，让窗口重新变成无重复字符的状态。

哈希表 `last` 用来记录每个字符上一次出现的位置。

关键点：

```python
if ch in last and last[ch] >= left:
    left = last[ch] + 1
```

其中 `last[ch] >= left` 表示当前字符上一次出现的位置还在窗口内，只有这种情况下才需要移动 `left`。

### 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last = {}
        left = 0
        ans = 0

        for right, ch in enumerate(s):
            if ch in last and last[ch] >= left:
                left = last[ch] + 1

            last[ch] = right
            ans = max(ans, right - left + 1)

        return ans
```

### 易错点

对于字符串：

```python
s = "abba"
```

遍历到最后一个 `'a'` 时，`'a'` 上一次出现的位置是 `0`，但此时窗口左边界已经在 `2`，所以这个 `'a'` 已经不在当前窗口中，不能让 `left` 回退。

因此必须判断：

```python
last[ch] >= left
```

否则会错误地把窗口左边界往回移动。

## 438. 找到字符串中所有字母异位词

### 核心思路

使用 **固定长度滑动窗口 + 字符频次数组 / 哈希表**。

异位词的本质是：两个字符串中每个字符出现次数完全相同。

因此可以维护一个长度为 `len(p)` 的窗口，每次判断当前窗口中的字符频次是否和 `p` 的字符频次相同。

这题的窗口长度固定，因为异位词必须和 `p` 长度相同。

### Counter 写法

```python
from collections import Counter

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        n, m = len(s), len(p)
        if n < m:
            return []

        p_count = Counter(p)
        window = Counter()
        ans = []

        for right, ch in enumerate(s):
            window[ch] += 1

            if right >= m:
                left_ch = s[right - m]
                window[left_ch] -= 1

                if window[left_ch] == 0:
                    del window[left_ch]

            if right >= m - 1 and window == p_count:
                ans.append(right - m + 1)

        return ans
```

### 数组写法

如果题目限定只包含小写英文字母，可以使用长度为 `26` 的数组代替 `Counter`。

```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        n, m = len(s), len(p)
        if n < m:
            return []

        p_count = [0] * 26
        window = [0] * 26
        ans = []

        for ch in p:
            p_count[ord(ch) - ord('a')] += 1

        for right, ch in enumerate(s):
            window[ord(ch) - ord('a')] += 1

            if right >= m:
                left_ch = s[right - m]
                window[ord(left_ch) - ord('a')] -= 1

            if right >= m - 1 and window == p_count:
                ans.append(right - m + 1)

        return ans
```

### 关键数据结构说明

`Counter` 可以理解为一个“计数字典”。

例如：

```python
from collections import Counter

window = Counter()
window['a'] += 1
window['b'] += 1
window['a'] += 1
```

此时：

```python
Counter({'a': 2, 'b': 1})
```

表示：

```text
'a' 出现 2 次
'b' 出现 1 次
```

### 为什么要删除计数为 0 的字符

窗口左边字符移出后：

```python
window[left_ch] -= 1
```

如果它的次数变成 `0`，说明这个字符已经不在当前窗口中，应该删除：

```python
if window[left_ch] == 0:
    del window[left_ch]
```

否则 `window` 可能残留类似下面的数据：

```python
Counter({'b': 1, 'c': 1, 'a': 0})
```

这不是真实窗口的干净表示，会影响后续和 `p_count` 的比较逻辑，也容易造成理解混乱。

## 560. 和为 K 的子数组

### 核心思路

使用 **前缀和 + 哈希表**。

定义当前前缀和为 `pre`，表示从数组开头到当前位置的元素和。

如果存在一个之前的前缀和 `old_pre`，满足：

```text
pre - old_pre = k
```

那么说明从 `old_pre` 后面一个位置到当前位置的这段子数组和为 `k`。

变形可得：

```text
old_pre = pre - k
```

所以遍历数组时，只需要查看之前有多少个前缀和等于 `pre - k`，就说明有多少个以当前位置结尾、和为 `k` 的子数组。

### 普通字典写法

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        count = {0: 1}
        pre = 0
        ans = 0

        for num in nums:
            pre += num

            if pre - k in count:
                ans += count[pre - k]

            count[pre] = count.get(pre, 0) + 1

        return ans
```

### defaultdict 写法

```python
from collections import defaultdict

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        count = defaultdict(int)
        count[0] = 1

        pre = 0
        ans = 0

        for num in nums:
            pre += num

            ans += count[pre - k]

            count[pre] += 1

        return ans
```

### 为什么不是滑动窗口

这题数组中可能包含负数。

滑动窗口通常适合窗口和具有单调性的情况，比如全是正数时，右边界右移会让窗口和变大，左边界右移会让窗口和变小。

但如果有负数，窗口和可能变大也可能变小，双指针无法稳定判断该移动哪一边。

因此这题更适合用前缀和。

### 为什么 `count[0] = 1`

`count[0] = 1` 表示：还没有遍历任何元素时，前缀和为 `0` 出现过一次。

它用于处理从下标 `0` 开始的子数组。

例如：

```python
nums = [1, 2, 3]
k = 3
```

遍历到 `2` 时：

```python
pre = 3
pre - k = 0
```

此时说明 `[1, 2]` 这个从开头开始的子数组和为 `3`。

如果没有提前记录 `count[0] = 1`，这个答案就会漏掉。

### `count[pre] = count.get(pre, 0) + 1` 的含义

这句表示：当前前缀和 `pre` 又出现了一次。

```python
count[pre] = count.get(pre, 0) + 1
```

其中：

```python
count.get(pre, 0)
```

表示：

- 如果 `pre` 已经存在，就取出它原来的出现次数；
- 如果 `pre` 不存在，就当作出现次数为 `0`。

等价写法：

```python
if pre in count:
    count[pre] += 1
else:
    count[pre] = 1
```

### defaultdict 的注意点

下面这种写法是错误的：

```python
count = defaultdict()
```

因为没有指定默认值工厂，访问不存在的 key 时不能自动补 `0`。

应该写成：

```python
count = defaultdict(int)
```

`int()` 的默认值是 `0`，因此：

```python
count[pre] += 1
```

在 `pre` 不存在时也不会报错。
