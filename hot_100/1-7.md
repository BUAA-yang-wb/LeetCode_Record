# LeetCode 热题刷题笔记：前 7 题

## 1. 两数之和

### 代码

```python
from typing import List

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        seen = {}

        for i, num in enumerate(nums):
            need = target - num

            if need in seen:
                return [seen[need], i]

            seen[num] = i
```

### 简要解析

用哈希表 `seen` 存储已经遍历过的数字及其下标：

```text
数字 -> 下标
```

遍历数组时，当前数字是 `num`，需要找的另一个数是：

```text
target - num
```

如果这个数已经在 `seen` 中，说明找到答案，直接返回两个下标。

### 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(n)
```

## 2. 字母异位词分组

### 代码

```python
from typing import List
from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)

        for s in strs:
            key = ''.join(sorted(s))
            groups[key].append(s)

        return list(groups.values())
```

### 简要解析

字母异位词排序后结果相同。

例如：

```text
"eat" -> "aet"
"tea" -> "aet"
"ate" -> "aet"
```

所以可以用排序后的字符串作为 key，用 `defaultdict(list)` 进行分组：

```text
排序后的字符串 -> 原字符串列表
```

### 复杂度

设字符串数量为 `n`，字符串平均长度为 `k`。

```text
时间复杂度：O(n * k log k)
空间复杂度：O(n * k)
```

## 3. 最长连续序列

### 代码

```python
from typing import List

class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        num_set = set(nums)
        longest = 0

        for num in num_set:
            if num - 1 not in num_set:
                current = num
                length = 1

                while current + 1 in num_set:
                    current += 1
                    length += 1

                longest = max(longest, length)

        return longest
```

### 简要解析

用 `set` 存储所有数字，方便 `O(1)` 判断某个数字是否存在。

只从连续序列的起点开始找：

```text
如果 num - 1 不在集合中，说明 num 是起点
```

然后不断判断 `num + 1` 是否存在，统计连续长度。

### 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(n)
```

## 4. 移动零

### 代码

```python
from typing import List

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0

        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

### 简要解析

用双指针：

```text
fast：负责遍历数组
slow：表示下一个非零元素应该放的位置
```

当 `fast` 遇到非零元素时，就把它交换到 `slow` 位置，然后 `slow += 1`。

这样可以保证所有非零元素相对顺序不变，且原地修改数组。

### 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(1)
```

## 5. 盛最多水的容器

### 代码

```python
from typing import List

class Solution:
    def maxArea(self, height: List[int]) -> int:
        left = 0
        right = len(height) - 1
        ans = 0

        while left < right:
            width = right - left
            h = min(height[left], height[right])
            ans = max(ans, width * h)

            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return ans
```

### 简要解析

容器面积由宽度和较短的柱子决定：

```text
面积 = (right - left) * min(height[left], height[right])
```

使用双指针，从最宽的情况开始，每次移动较短的一边。

原因是：面积受短板限制，移动较高的一边没有意义，只有移动较短的一边才可能找到更大的面积。

### 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(1)
```

## 6. 三数之和

### 代码

```python
from typing import List

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        ans = []
        n = len(nums)

        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left = i + 1
            right = n - 1

            while left < right:
                total = nums[i] + nums[left] + nums[right]

                if total == 0:
                    ans.append([nums[i], nums[left], nums[right]])

                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1

                    left += 1
                    right -= 1

                elif total < 0:
                    left += 1
                else:
                    right -= 1

        return ans
```

### 简要解析

先排序，然后固定第一个数 `nums[i]`，在右侧用双指针找另外两个数。

如果三数之和：

```text
等于 0：加入答案，并跳过重复元素
小于 0：left 右移，让和变大
大于 0：right 左移，让和变小
```

排序的作用：

```text
1. 方便使用双指针
2. 方便去重
```

### 复杂度

```text
时间复杂度：O(n^2)
空间复杂度：O(1)，不算返回结果
```

## 7. 接雨水

### 代码

```python
from typing import List

class Solution:
    def trap(self, height: List[int]) -> int:
        left = 0
        right = len(height) - 1

        left_max = 0
        right_max = 0
        ans = 0

        while left < right:
            if height[left] < height[right]:
                if height[left] >= left_max:
                    left_max = height[left]
                else:
                    ans += left_max - height[left]
                left += 1
            else:
                if height[right] >= right_max:
                    right_max = height[right]
                else:
                    ans += right_max - height[right]
                right -= 1

        return ans
```

### 简要解析

每个位置能接多少水，取决于左右两边最高柱子的较小值：

```text
当前位置接水量 = min(左边最高, 右边最高) - 当前高度
```

用双指针维护：

```text
left_max：左侧目前最高柱子
right_max：右侧目前最高柱子
```

每次处理较矮的一侧。如果当前高度小于对应方向的最大高度，就可以接水；否则更新最大高度。

### 复杂度

```text
时间复杂度：O(n)
空间复杂度：O(1)
```

## 常见数据结构和语法总结

### dict 字典 / 哈希表

适合存对应关系：

```python
seen = {
    2: 0,
    7: 1
}
```

含义：

```text
数字 -> 下标
```

常见场景：两数之和。

---

### set 集合 / 哈希集合

适合快速判断元素是否存在：

```python
num_set = set(nums)
```

常见场景：最长连续序列。

---

### defaultdict(list)

本质是字典，但 key 不存在时会自动创建空列表。

适合分组：

```python
groups[key].append(value)
```

常见场景：字母异位词分组。

---

### enumerate

不是数据结构，而是遍历工具。

可以同时拿到下标和值：

```python
for i, num in enumerate(nums):
    ...
```

等价于：

```python
for i in range(len(nums)):
    num = nums[i]
```

---

### 双指针

常见形式：

```text
left / right
slow / fast
```

适合数组原地操作、有序数组查找、区间问题。

常见题目：

```text
移动零
盛最多水的容器
三数之和
接雨水
```
