## 189. 轮转数组

### 代码

```python
class Solution:
    def rotate(self, nums: list[int], k: int) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        k %= n

        def reverse(left: int, right: int) -> None:
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1

        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
```

### 简要分析

使用三次翻转法。

以 `nums = [1,2,3,4,5,6,7]`，`k = 3` 为例：

```text
原数组：       [1,2,3,4,5,6,7]
整体翻转：     [7,6,5,4,3,2,1]
翻转前 k 个：  [5,6,7,4,3,2,1]
翻转剩余部分： [5,6,7,1,2,3,4]
```

注意 `k` 可能大于数组长度，所以需要先执行：

```python
k %= len(nums)
```

## 238. 除自身以外数组的乘积

### 代码

```python
class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        n = len(nums)
        answer = [1] * n

        left_product = 1
        for i in range(n):
            answer[i] = left_product
            left_product *= nums[i]

        right_product = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right_product
            right_product *= nums[i]

        return answer
```

### 简要分析

对每个位置 `i` 来说，答案等于：

```text
nums[i] 左边所有元素的乘积 × nums[i] 右边所有元素的乘积
```

第一遍从左往右，先把每个位置左边的乘积存入 `answer`。

以 `nums = [1,2,3,4]` 为例，第一遍后：

```text
answer = [1,1,2,6]
```

第二遍从右往左，把右边的乘积乘进 `answer`。

```text
最终结果：[24,12,8,6]
```

这个方法不需要使用除法，也可以处理数组中有 `0` 的情况。

## 41. 缺失的第一个正数

### 代码

```python
class Solution:
    def firstMissingPositive(self, nums: list[int]) -> int:
        n = len(nums)

        for i in range(n):
            while 1 <= nums[i] <= n and nums[nums[i] - 1] != nums[i]:
                correct_index = nums[i] - 1
                nums[i], nums[correct_index] = nums[correct_index], nums[i]

        for i in range(n):
            if nums[i] != i + 1:
                return i + 1

        return n + 1
```

### 简要分析

核心思想是原地哈希：把数字 `x` 放到下标 `x - 1` 的位置。

对应关系是：

```text
1 应该放在 nums[0]
2 应该放在 nums[1]
3 应该放在 nums[2]
...
```

以 `nums = [3,4,-1,1]` 为例：

```text
3 应该去下标 2
4 应该去下标 3
1 应该去下标 0
```

整理后大致变成：

```text
[1,-1,3,4]
```

然后从左到右检查，第一个不满足 `nums[i] == i + 1` 的位置就是答案。

这里 `nums[1] != 2`，所以返回 `2`。

注意 `while` 中需要判断：

```python
nums[nums[i] - 1] != nums[i]
```

这是为了避免重复数字导致死循环，比如 `[1,1]`。
